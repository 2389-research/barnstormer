Process

From idea to spec
in four moves.

No templates. No blank pages. Just describe your vision and let the swarm take it from there.

01

Describe

Tell barnstormer what you want to build. A sentence, a paragraph, or a brain dump. The Manager agent parses it into structured fields.

02

Swarm

Five agents activate. They brainstorm ideas, organize plans, identify risks, and map relationships — running concurrently in the background.

03

Refine

Agents ask you questions when they need human judgment. Answer in the chat rail, and the swarm incorporates your guidance instantly.

04

Export

Get your finished spec as Markdown, YAML, or DOT. Cards flow through lanes — Ideas, Plan, Spec — until everything is nailed down.

Capabilities

Engineered for
serious specs.

Not another chatbot wrapper. barnstormer is an event-sourced system built with Rust, designed for durability and real-time collaboration.

Core

Event Sourcing

Every mutation is a command that produces an event. State is materialized by replaying the event log. Full audit trail, time travel, and undo built in.

UI

Real-Time Web UI

HTMX + SSE streaming. No page reloads. Board, document, and diagram views update live as agents work.

LLM

Multi-Provider

Anthropic, OpenAI, or Gemini. Switch providers with an environment variable. Auto-detects available API keys.

Board

Kanban Board

Drag-and-drop cards across lanes. Ideas flow to Plans, Plans become Spec. Agents and humans move cards together.

Viz

DOT Diagrams

Architecture diagrams auto-generate from card relationships. Rendered with Viz.js. Copy or download the DOT source.

Durability

Crash Recovery

JSONL event logs are the source of truth. SQLite provides a queryable cache. Snapshots are saved automatically, and full state recovers on restart.

The Swarm

Five agents.
One specification.

Each agent has a focused role, its own event receiver, and a set of tools for reading state, writing commands, and communicating with you.

Coordinator

Manager

Primary point of contact. Parses your description, creates initial cards, asks clarifying questions, and coordinates the swarm.

Researcher

Brainstormer

Generates creative ideas and explores possibilities. Creates idea cards with breadth-first exploration and narrates its thinking.

Architect

Planner

Organizes ideas into structured plans. Moves promising ideas forward, creates task cards, and defines constraints and success criteria.

Analyst

Diagram

Analyzes spec structure and card relationships. Identifies gaps, suggests improvements, and summarizes pipeline health.

Reviewer

Critic

Reviews for gaps, inconsistencies, and risks. Creates risk and constraint cards. Asks about ambiguities needing human judgment.

System

Event-sourced
from the ground up.

Commands produce events. Events build state. State drives the UI. The JSONL log is the single source of truth.

Quick Start

Up and running
in sixty seconds.

terminal
# Clone and build
git clone https://github.com/your-org/barnstormer
cd barnstormer
cargo build --all

# Configure an LLM provider
cp .env.example .env
# Edit .env — set ANTHROPIC_API_KEY, OPENAI_API_KEY, or GEMINI_API_KEY

# Launch (opens browser at 127.0.0.1:7331)
cargo run -- start
